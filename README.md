# 学生成绩管理系统

一个用 Python 写的命令行学生成绩管理工具，支持增删改查、数据统计、文件导入导出等功能。

---

## ✨ 功能列表

- 添加学生、添加成绩、删除学生、修改成绩
- 显示所有学生、查看学生平均分、按科目查询、模糊搜索
- 班级成绩统计、各科目平均分、按分数段筛选、总分排名
- 从 CSV 文件导入成绩、导出学生名单
- 自动保存到文件，下次启动自动加载
- 重置所有数据、查看缺考学生、各科目最高/最低分

---

## 🚀 如何运行

1. 确保已安装 Python 3
2. 在终端运行：

python grade_manager.py
---

## 📥 导入示例文件

创建一个 `grades.csv` 文件，内容如下：

姓名,科目,分数
张三,数学,85
张三,英语,90
李四,数学,78
李四,英语,88
